# Krasjkurs-OOP-2023

Velkommen til kræsjkurs i objektorientert programmering!

Jeg vet ikke hva som kommer på eksamen, men kurset vil gå gjennom temaer som pleier å komme på eksamen og hva jeg tror kan komme på eksamen. I tillegg vil jeg forklare litt rundt de ulike konseptene innen objektorientert programmering og belyse typiske feller som kan komme på eksamen.

I mappen eksamensforelesning ligger filene som vil programmeres i underveis i forelesningen. EksamensforelesningLF inneholder de samme filene, men med løsninger på problemene. Koden er en blanding av gamle eksamensoppgaver og egne tester for å belyse sentrale deler av emnet.

Temaer som vil gås gjennom er:
- Delegering
- Arv
- Observatør/Observert
- Grensesnitt
- Funksjonelle grensesnitt
    - Predikater
    - Iterator/Iterable
    - Comparator/Comparable
- Streams
- Filbehandling
- Testing
    - Koden som brukes til testing ligger ikke i dette repoet. Oppgaven som vil gjennomgås er MedianComputerTest fra eksamen 2020.

Vel møtt!
